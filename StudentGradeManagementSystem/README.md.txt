# Student Grade Management System (C# WinForms)

## Overview
This is a Windows Forms application developed in C# that allows users to manage student grades.

## Features
- Add Student
- Display Students
- Search Student
- Calculate Average Grade
- Highest & Lowest Grades
- Grade Categories using Enum
- Exception Handling

## How to Run
1. Open solution in Visual Studio
2. Build
3. Run
